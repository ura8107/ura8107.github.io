駒場国際政治ワークショップについて

駒場国際政治ワークショップは、東京大学大学院総合文化研究科国際社会科学専攻で国際政治を学ぶ大学院生が運営する研究会です。本ワークショップは、駒場における国際政治学・国際政治経済学の議論を活性化させるとともに、研究者ネットワークの構築・促進を図ることを目的としています。また、駒場にはかねてから所属ゼミの垣根を越えた学術交流に対する要望があり、それに応える形で発足しました。顧問の石田淳先生・湯川拓先生・吉本郁先生にご協力いただきながら、2009年度より開催しています。ワークショップは各学期に3回ずつ、1年度で計6回開催しています。過去の開催内容は開催実績、ワークショップでの報告に関わる刊行論文・書籍については関連刊行物をご覧ください。
